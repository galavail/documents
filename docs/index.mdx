Cruise Booking Platform (eCommerce) de GalAvail y tiene como fin facilitar la integración del motor de búsqueda en sus sitios web.
El buscador permite personalizar consultas mediante parámetros en la URL y mostrar disponibilidad actualizada directamente desde el inventario centralizado de cruceros.



## Parámetros de la URL de Búsqueda

La URL de búsqueda está compuesta por 15 parámetros que deben ser configurados según las especificaciones de cada uno. El formato estándar de la URL es:

```
https://agencies.galavail.com/companyName/search/adults/children/months/year/priceFrom/priceTo/cruiseType/cruiseLength/cruiseName/rateType/sortBy/cruiseCategories
```

> **Nota:** Si el parámetro “Rate Type” es “Charter Only”, la URL debe terminar con `/charter`.

### Tabla de Parámetros

| Orden | Componente URL      | Descripción                        |
|-------|---------------------|------------------------------------|
| 1     | baseURL             | Principal URL                      |
| 2     | companyName         | Name of the company                |
| 3     | search              | Search parameter                   |
| 4     | adults              | Number of adults                   |
| 5     | children            | Number of children                 |
| 6     | months              | Travel month                       |
| 7     | year                | Travel year                        |
| 8     | priceFrom           | Price from                         |
| 9     | priceTo             | Price up                           |
| 10    | cruiseType          | Type of cruise                     |
| 11    | cruiseLength        | Tour duration                      |
| 12    | cruiseName          | Cruise name                        |
| 13    | rateType            | Rate type                          |
| 14    | sortBy              | Sort by                            |
| 15    | cruiseCategories    | Cruise category                    |

### Principal URL

Url principal del eCommerce de Agencias

```
https//agencies.galavail.com 

```
<Info>Este parámetro no es configurable</Info>


### Name of the company

Nombre de la compania proporcionado por Galavail

```
companyName
```
<Info>Este parámetro es único por cliente de la plataforma eCommerce, se debe remplazar la palabra **companyName** por el proporcionado por Galavail.</Info>

### Search parameter
Conformado por la palabra reservada **search** como parte de la URL.
```
search
```
<Info>Parámetro no configurable, forma parte de la url principal del eCommerce del cliente</Info>

### Number of adults
Indica el número de personas adultas que se considere en la búsqueda.

<Card title="Rango de Valores" icon="table-list">
  El valor permitido es de **1** persona hasta **20** personas.

    | Valor Parámetro   | Descripción               |
    |------------------ |---------------------------|
    | 1                 | 1 Persona                 |
    | 2                 | 2 Personas                |
    | 3                 | 3 Personas                |
    | ...               | ..                        |
    | 20                | 20 Personas               |
</Card>


### Number of children
Indica el número de niños que se considere en la búsqueda.

<Card title="Rango de Valores" icon="table-list">
  El valor permitido es de **0** niños hasta **10** niños.

    | Valor Parámetro   | Descripción               |
    |------------------ |---------------------------|
    | 0                 | No existen niños          |
    | 1                 | 1 niños                   |
    | 2                 | 2 niños                   |
    | ...               | ..                        |
    | 10                | 10 niños                  |
</Card>
